# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** verbatim. The full text is published at the canonical source:

➡️ <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

It applies to all project spaces — issues, pull requests, Discussions, commits, and any other communication using project infrastructure.

## Reporting

If you observe behavior that violates the Code of Conduct, contact the maintainer privately. Use one of:

- Email the maintainer (see the `author` field in `package.json`) with `[tsfix-conduct]` in the subject line.
- Open a private message via the GitHub profile of an active maintainer.

Reports stay confidential. We respond within 5 business days, investigate, and take action proportional to the report. Retaliation against reporters is itself a violation.

## Why a pointer rather than embedding the full text

The Contributor Covenant is a living document — the upstream version is the authoritative one, and pinning a copy here risks drift if version 2.2 (or later) supersedes it. Linking to the canonical source guarantees you always read the current text.
